# Шаблон педагогического сайта в стиле клейморфизм

Яркий и мягкий шаблон для педагога (воспитателя, учителя начальных классов, педагога допобразования) с дружелюбной клейморфной эстетикой. Подходит детям 4–10 лет, родителям и коллегам.

## Как пользоваться
1. Откройте `index.html` в браузере или разместите файлы на любом статическом хостинге.
2. Замените текстовые блоки и ссылки на свои данные (имя, курсы, материалы, соцсети).
3. При необходимости поменяйте цвета в корневых CSS-переменных (файл `styles.css`, блок `:root`).
4. Для быстрых иконок можно использовать emoji или подключить набор иконок (например, [Phosphor Icons](https://phosphoricons.com)).

## Как быстро проверить и выложить на GitHub Pages

### Локально
1. Скачайте репозиторий или архив с файлами.
2. Откройте `index.html` двойным кликом в браузере **или** запустите простой сервер:
   ```bash
   # внутри папки проекта
   python3 -m http.server 8000
   # откройте http://localhost:8000 в браузере
   ```

### GitHub Pages (бесплатная публикация со ссылкой)
1. Создайте новый публичный репозиторий на GitHub и загрузите туда `index.html` и `styles.css`.
2. В настройках репозитория откройте **Settings → Pages**.
3. В секции **Build and deployment** выберите **Deploy from a branch**.
4. Укажите ветку `main` и папку `/ (root)`, сохраните настройки.
5. Через ~1–2 минуты появится ссылка вида `https://<ваш-ник>.github.io/<имя-репозитория>/` — это готовый сайт.

### Обновления
Изменили тексты или цвета? Просто закоммитьте обновления в ветку `main` — GitHub Pages автоматически пересоберёт сайт за пару минут.

## Структура страницы
- **Главная (hero)**: приветствие, имя, слоган, CTA.
- **О педагоге**: фото-заглушка, подход, основные направления.
- **Занятия / курсы**: карточки с описанием и кнопкой действия.
- **Материалы для детей**: блоки для игр, заданий, раскрасок (ссылки «Скачать»).
- **Для родителей**: советы и принципы работы.
- **Контакты**: форма обратной связи и кнопки соцсетей.
- **Футер**: короткое позиционирование.

## Визуальный стиль
- **Клейморфизм**: мягкие объемные карточки, внутренние и внешние тени, плавные градиенты.
- **Форма**: крупные радиусы (26px), закругленные кнопки и карточки.
- **Тени**: комбинированные внешние и внутренние тени для «пластилинового» эффекта.
- **Иконки**: дружелюбные emoji или кастомные иллюстрации.

## Цветовая палитра (pastel / candy)
- Голубой: `#9fd6ff`
- Розовый: `#f7b8d6`
- Мята: `#b4f1e0`
- Желтый: `#ffe8a3`
- Лаванда: `#dcd0ff`
- Фон: `#f4f7ff`
- Текст: `#3a3f52`, доп. текст `#6d7289`

## Рекомендации по шрифтам
- **Дисплей**: [Baloo 2](https://fonts.google.com/specimen/Baloo+2) — мягкий, дружелюбный заголовочный.
- **Текст**: [Nunito](https://fonts.google.com/specimen/Nunito) — читабельный на мобильных.

## UX-настройки
- Крупные кликабельные зоны (`padding` 14–18px на кнопках).
- Контрастные, но мягкие CTA (градиентные кнопки `primary`).
- Сетки, которые автоматически перестраиваются под ширину экрана (`auto-fit` и `minmax`).
- Короткие тексты и маркированные списки вместо длинных абзацев.

## Быстрый чек-лист адаптации под себя
- Заменить имя и слоган в блоке hero.
- Вставить реальное фото (класс `.avatar`) или добавить `<img>`.
- Подставить свои курсы и материалы (карточки в секциях `#courses` и `#materials`).
- Обновить контакты и ссылки на соцсети.
- При желании настроить цвета через CSS-переменные или добавить новые тени.
